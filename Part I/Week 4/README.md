# Week 4 - Control Flow: Decision Making and Looping

This directory contains exercises and examples for the fourth week of the C programming journey. During this week, you will learn about control flow statements, including decision-making constructs and looping structures. These constructs enable you to control the execution flow of your programs.

Topics covered during this week include:
- Conditional statements: if, if-else, switch
- Looping statements: while, do-while, for
- Nested loops and loop control statements
- Break and continue statements
